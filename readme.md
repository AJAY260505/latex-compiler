 # 🌟 LaTeX Compiler Web App 🌟

```markdown

Welcome to the LaTeX Compiler Web App! This simple  web application allows you to **compile LaTeX code into PDF documents** with ease. Just enter your LaTeX code, hit **compile**, and voilà—your PDF is ready to download!

## 🚀 Features

- **Compile LaTeX code** into high-quality PDF documents.
- **Real-time preview** of the compiled document.
- Built with **Docker** for easy local development and deployment.
- Optimized for simplicity and speed.

## 💻 How to Run Locally

Follow these easy steps to get the app running on your local machine:

1. Clone the repository:

   ```bash
   git clone https://github.com/ajay260505/latex-compiler.git
   cd latex-compiler-app
   ```

2. Build and run the app using Docker Compose:

   ```bash
   docker-compose up --build
   ```

3. Open your browser and go to `http://localhost:3000`. 🎉 You’re all set!

## 🌐 Deployment

Want to deploy your app online? You can easily host it on free platforms like [Render](https://render.com) or [Railway](https://railway.app). Just follow their simple deployment guides for Docker-based applications.
